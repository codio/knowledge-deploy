Blackboard LTI 1.x¶

Please be sure to check out the Codio LTI App which allows for an easy way to integrate and to map Codio course assignments to your LMS system. The following page explains how to set up external apps in Blackboard Learn.

In Codio:

Enable LTI for Your Course¶

  1. Open the course you would like to connect or create a new course.

  2. Make sure you have at least one published assignment or add a new one. (see Add and Remove Course Assignments)

  3. Select the LTI/LMS tab.

  4. Select the ENABLE LTI option.

  1. Click Save Changes.

Bring up the LTI Integration Information¶

  1. Click your user name in the bottom left of your dashboard

  2. Choose your Organization

  3. Click the LTI Integrations tab to bring up the following settings.

In Blackboard¶

The Blackboard user who carries out these steps must be a Blackboard system administrator.

  1. Create a new Course in Blackboard. We suggest you create a test course called Codio Blackboard before you do it with a production course.

  2. Look for the System Admin tab near the top right of the page and select it.

  3. Look for the Building Blocks section and select it.

  4. Click on LTI tool providers.

  5. Click on Register Provider Domain in the menu bar.

  6. In the Provider Domain field, enter apollo.codio.com.

  7. In the Default Configuration section, set Default Configuration to be Set globally.

  8. In the Organization Policies section you should

  9. set Send User Data to Send user data only over SSL.

  10. in User Fields to Send you should set all 3 fields (Constituency in Course, Name, Email Address).

In Codio and Blackboard¶

  1. Now return to the Default Configuration section in Blackboard. We will now copy the following global integration fields from Codio to Blackboard.

  • LTI Consumer -> Tool Provider Key

  • LTI Secret -> Tool Provider Secret

Codio course setup¶

You need to perform the following actions one time only for a course. The Blackboard user who carries out these steps does not need to be a system administrator but must have suitable privileges to edit courses and assignments.

  1. In Codio, go to your course and click on the LTI/LMS tab.

  2. Go to the LTI/LMS Settings area.

  3. At the top is a switch Enable LTI which you should check is enabled.

  4. Below this is an empty field Course LMS URL. Switch back to Blackboard and make sure you are on the main the Codio Blackboard course you created earlier. Copy the url in the address bar of your browser to the clipboard and paste it into the above mentioned field in Codio.

This URL ensures that Codio knows how to redirect students back to the correct Blackboard course should they attempt to access the course through their dashboard.

Mapping an assignment to Blackboard Content¶

The final mapping step needs to be taken for each individual assignment within Codio. It maps a piece of Blackboard content to a Codio assignment.

In Codio¶

  1. On the main course screen, make sure the Edit Assignments tab is selected.

  2. Click the Add Assignment button and select Project Based.

  3. Select a project that has some autograded assessments. The My First Project that you created earlier in the Onboarding Guide has some auto-graded assessments. You can also assign another project but you will need to manually grade the assignment so there are some scores to pass back to the Blackboard gradebook.

  4. Once the assignment has been added to the course, you should click the icon with 3 blue dots and select LTI Integration URL.

  5. You should copy the LTI integration url to the clipboard by clicking on the field (it will auto copy).

In Blackboard¶

We now return to Blackboard complete the mapping.

  1. Make sure you have selected the Blackboard course.

  2. Click Content in the left hand menu.

  3. Select or hover over Build Content in the menu bar

  4. Select Web Link.

  5. In the Web Link Information section …

  6. Enter a name for the content

  7. Paste in the LTI Integration URL that you just copied to the clipboard from Codio

  8. Check the box Ths is a link to a tool provider

  9. Select Yes for the field Enable Evaluation after which more fields will appear

  10. Set the points you want to award for this content (Codio will automatically scale the percentage value it uses to the points you specify here)

  11. Save the content settings.
